Class ClientBase

Namespace
Ebceys.Infrastructure.HttpClient
Assembly
Ebceys.Infrastructure.dll

Abstract base class for HTTP clients built on Flurl. Provides standardized methods for GET, POST, PUT, DELETE requests with JSON serialization/deserialization, automatic authorization header injection, Polly retry policies, and structured logging of requests and responses.
