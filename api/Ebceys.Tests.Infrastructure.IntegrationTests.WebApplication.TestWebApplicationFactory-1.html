Abstract test web application factory that creates an in-memory test server for integration testing. Extends WebApplicationFactory<TEntryPoint> with support for Serilog logging, optional production appsettings loading, and automatic execution of IBeforeHostingStartedService implementations during host creation.

[PublicAPI]
public abstract class TestWebApplicationFactory<TStartup> : WebApplicationFactory<TStartup>, IDisposable, IAsyncDisposable where TStartup : class
