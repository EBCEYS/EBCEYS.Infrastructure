Namespace Ebceys.Tests.Infrastructure.Helpers.TestCaseSources

Classes

EbTestCaseSource<TData>

Abstract base class for NUnit parameterized test case sources. Provides a pattern for defining valid and invalid test data generators with a shared base data setup. Used with EbTestDataValidExecutor<TEnumerator, TData> and EbTestDataInvalidExecutor<TEnumerator, TData>.

EbTestDataInvalidExecutor<TEnumerator, TData>

NUnit test case enumerator that generates invalid test cases from EbTestCaseSource<TData>. Yields all cases returned by GetInvalidCases() using the invalid base data.

EbTestDataValidExecutor<TEnumerator, TData>

NUnit test case enumerator that generates valid test cases from EbTestCaseSource<TData>. Yields the valid base data plus all cases returned by GetValidCases().

TestCaseEnumerator<TData>

Abstract base class for NUnit NUnit.Framework.TestCaseData enumerators that converts test case transformation functions into NUnit.Framework.TestCaseData instances.
